# Student Management System

## Description
A C++ console application to manage students and their courses.  
This system allows adding, deleting, and viewing students, as well as managing the courses each student is enrolled in.  
Demonstrates Object-Oriented Programming (OOP) concepts including classes, encapsulation, and dynamic data storage with vectors.

## Features
- Add a new student
- Delete an existing student
- Add a course to a student
- Delete a course from a student
- Display all students and their courses
- Input validation and error handling
- Console-based interactive menu

## Installation
1. Clone the repository
2. Compile the code using a C++ compiler
3. Run the program

## Usage
- Follow the on-screen menu to perform operations.
- Enter the corresponding number to select an action.
- Enter the requested information (ID, name, grade, course) as prompted.

## Technologies & Skills
- C++ (OOP)
- Classes and encapsulation
- Vectors for dynamic storage
- Console input/output
